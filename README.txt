PROJETO — MODELO DE TESE UFPI

Este projeto reúne materiais de apoio para uso de um modelo de tese em LaTeX, com foco em organização, instalação do ambiente e escrita acadêmica.

O que há no projeto:
- modelo principal da tese;
- preâmbulo e estrutura inicial do documento;
- capa e elementos pré-textuais;
- capítulos de exemplo;
- arquivo de referências bibliográficas;
- manual de instalação e configuração do ambiente;
- arquivos auxiliares para instalação;
- recomendação prática sobre uso de script para melhorar a digitação em matemática.

Sumário estrutural:
1. Modelo da tese
2. Estrutura do documento
3. Referências bibliográficas
4. Manual de instalação
5. Arquivos auxiliares
6. Material complementar de produtividade

Objetivo:
Oferecer ao leitor uma visão rápida da estrutura do projeto e dos materiais disponíveis.

Aproveite!
